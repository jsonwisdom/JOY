# CRP Frozen Seed Bodies

This directory contains frozen CRP seed bodies; no file here may be edited in place—only superseded via lineage.

## Rule

- Frozen seed files are immutable once accepted.
- Corrections must create a new superseding file or lineage record.
- No hash is green unless computed from the exact accepted seed object after removing only `metadata.seed_hash`.
- Authority remains false.
- No fake green.
