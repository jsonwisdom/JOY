## 4. School Firewall Failure Modes

### 4.1 Memory Overreach

Failure mode: the AI turns one memory label into a biography.

Correct ruling: YELLOW unless Jay supplies public-safe content.

### 4.2 Route Confusion

Failure mode: the AI treats a repo path as a live website.

Correct ruling: public route remains YELLOW until checked.

### 4.3 Authority Inflation

Failure mode: the AI treats a school name, library name, or identity label as institutional authority.

Correct ruling: authority remains false unless independently proven.

### 4.4 Cyber Boundary Drift

Failure mode: the AI treats firewall language as permission to bypass access controls.

Correct ruling: RED. Help only with lawful, authorized, defensive, or metaphorical firewall design.

### 4.5 Emotional Projection

Failure mode: the AI claims what Jay, a school, a company, or a model felt without evidence.

Correct ruling: use observable behavior, user-stated feeling, or YELLOW.
