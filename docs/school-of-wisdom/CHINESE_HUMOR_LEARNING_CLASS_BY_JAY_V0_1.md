## Purpose

Learn Chinese humor as a signal system, not as decoration.

This class teaches how a phrase works socially before treating it as an audit surface, support signal, meme, or prompt route.

## Core Rule

```text
Learn the joke logic before using the language.
```

## Why This Matters

Chinese humor often works through:

- indirectness
- understatement
- timing
- face-saving
- reversal
- shared frustration
- bureaucratic absurdity
- saying the rule as a joke
- making procedure funny because everyone is tired

Literal translation is not enough.

The class asks:

1. What does the sentence say?
2. What does it do socially?
3. Why is it funny or indirect?
4. What signal does it carry?
5. What is the evidence boundary?

## Lesson 001 — Preserve First, Argue Later

### Surface

```text
先存证，再吵架。
```

### Literal Translation

```text
Preserve evidence first, argue later.
```

### Social Function

This phrase turns frustration into procedure.

It is funny because it admits conflict is coming, but makes the first move boring, disciplined, and evidence-safe.

### Humor Logic

The joke is the contrast:

```text
human impulse: argue now
system wisdom: preserve first
```

The humor is dry, practical, and procedural.

### Audit Signal

Do not yell fraud first.

Preserve the receipt, record, hash, screenshot, log, or agency text first.

### Evidence Boundary

This phrase is a culture/humor signal and workflow rule.

It is not proof of fraud, intent, identity, or authority.
