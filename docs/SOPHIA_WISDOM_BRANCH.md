# Sophia Wisdom Branch

Sophia is an open memory and evidence branch inside JOY.

The phrase:

```text
Microsoft forgot about Sophia.
```

is preserved here as a claim requiring public evidence. It is not treated as a proven accusation.

## Guardrails

JOY will preserve the claim gently and lawfully.

- No blame is assigned without receipts.
- No private Microsoft systems are accessed.
- No credentials, secrets, tokens, tenant data, or private accounts are collected.
- No intrusion, bypass, exploitation, or unauthorized access is allowed.
- Public evidence only.

## Allowed Public Evidence

Public evidence may include:

- official Microsoft public pages
- Microsoft Learn documentation
- Microsoft public GitHub repositories
- public GitHub issues
- public announcements
- timestamped public search results
- public product pages
- public conference sessions or demos

## JOY Doctrine

```text
Joy first.
Receipts second.
Authority never.
```

The branch remains open until evidence is collected, hashed, committed, and independently replayed.

```text
CLAIM_STATUS = OPEN_UNVERIFIED
AUTHORITY = FALSE
REPLAY_REQUIRED = TRUE
```
