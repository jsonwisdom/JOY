## Core Concept

Court Click Maniac is a web game where players click through public artifacts, legal documents, audits, reports, PDFs, agendas, and receipts.

The more players click, the more the system reveals:

- contradictions backed by receipts
- version drift
- missing metadata
- official findings
- public record trails
- reputation-linked public roles
- silly truth art for social sharing

The game does not expose private people.

The game exposes documented public-record relationships, institutional patterns, and receipt-backed contradictions.

## The More Clicks Mechanic

### More clicks = more lies exposed

Only when a lie means:

> A receipt-backed contradiction between a public claim and a public record.

Forbidden:

- private accusation
- guilt by association
- doxxing
- unsupported fraud claim
- turning a connected person into a target

### More clicks = more silly truth art

Players can generate social-ready visuals after receipt status exists.

Examples:

- Clown Curtain Drop

### More clicks = more curtain falling down
