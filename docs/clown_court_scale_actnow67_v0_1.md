## ALMS Machine Speed Classification

ALMS does not decide truth.

ALMS classifies artifacts for replay.

```json
{
  "alms_classification": {
    "artifact_class": [
      "LOCAL_PUBLIC_RECORD",
      "FEDERAL_OVERSIGHT_REPORT",
      "LEGAL_AUTHORITY_TEXT",
      "AUDIT_REPORT",
      "COURT_RECORD",
      "OUTSIDE_PDF"
    ],
    "admission_level": [
      "OBSERVATION",
      "CLAIM",
      "RECEIPT",
      "VERIFIED_RECEIPT",
      "DECISION",
      "ARCHIVE"
    ],
    "forbidden_transition": "NO_SILENT_CATEGORY_PROMOTION"
  }
}
```

Machine speed is allowed for sorting, hashing, summarizing, indexing, and replay.

Machine speed is not allowed for accusation, guilt, intent, or authority.

## Outside PDFs and the Law

Outside PDFs enter only as artifacts.

A PDF may contain official findings, charges, pleas, convictions, recommendations, statutes, or legal text.

The system must preserve the difference between:

- what the document says
- who authored it
- what court or agency status it carries
- whether a claim is adjudicated
- whether Jay or any player is making a new claim

Rule:

> Reported official finding is not the same thing as new system authority.
