## Zora Interpretation

A Zora artifact may carry culture, timing, symbol, title, and public signal. Those surfaces matter. But an artifact does not become verified evidence merely because it exists on a public platform.

The replay lane keeps those domains separate:

```text
Symbolic artifact ≠ verified claim
Mint event ≠ institutional proof
Timestamp ≠ conclusion
Public visibility ≠ authority
Replay path ≠ identity fork
```

## Correct Function of Replay

Replay allows an observer to ask:

1. What was published?
2. When was it published?
3. What metadata was attached?
4. What transaction or content identifier supports the observation?
5. Can another observer follow the same path and reach the same state classification?

If yes, the observation may become replayable.

If no, it stays symbolic, reported, or claimed.
