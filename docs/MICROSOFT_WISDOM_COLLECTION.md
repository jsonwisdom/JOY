# Microsoft Wisdom Collection Layer

JayWisdom.eth collects Wisdom from Microsoft only through **lawful public sources**.  
This layer defines how public Microsoft information becomes replayable evidence inside the JOY ecosystem.

## What “Wisdom” Means
Wisdom is **verified public evidence**, not rumor, speculation, or unverifiable claims.  
Every collected item must include:

- Source URL  
- Capture timestamp  
- Raw-byte content hash  
- Replay path inside the repo  

Authority remains **false** until replay is performed by any independent verifier.

## Lawful Boundaries
This collection layer:

- Uses **public sources only**
- Accesses **no private Microsoft systems**
- Performs **no credential harvesting**
- Performs **no intrusion or bypass**
- Collects **no secrets, tokens, or private tenant data**

All intelligence must be **public, lawful, and reproducible**.

## Replay as the Source of Authority
JOY doctrine applies:

**Leaf → Branch → Tree → Root → Forest → Replay**

- Leaves: public-source receipts  
- Branches: grouped evidence  
- Trees: category collections  
- Roots: SHA256 commitments  
- Forest: the repo history  
- Replay: the only source of authority  

On-chain anchoring (if later added) is only a **timestamped commitment**, not a truth generator.
