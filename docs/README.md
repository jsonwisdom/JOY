# JOY GitHub Pages Draft

This directory contains the public reader draft for JOY.

## Enable GitHub Pages

1. Open the repository settings.
2. Go to Pages.
3. Select Deploy from branch.
4. Choose `main` and `/docs`.
5. Save.
6. Open the Pages URL and verify the page loads.

## State Boundary

```text
SITE_FILES_COMMITTED != LIVE_SITE_VERIFIED
LIVE_URL_LOADED != FAMILY_APPROVAL
GITHUB_GREEN != MRS_WISDOM_GREEN
NO_FAKE_GREEN_ACTIVE
```

## Public Safety

This site is symbolic and public-safe. It should not expose private family details or claim family approval before consent exists.
