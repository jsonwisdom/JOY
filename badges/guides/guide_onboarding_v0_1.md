## Purpose

This guide helps a new Joy Gardener enter JOYSPACE without needing to read the whole repository.

It is warm, simple, teen-safe, family-safe, and low pressure.

## What JOYSPACE Is

JOYSPACE is a safe digital garden for learning, creativity, kindness, truth, and growth.

It helps people build small things that make the future feel exciting, not frightening.

```text
Build a world where kindness is code and truth is open.
```

## First Three Steps

### 1. Look Around

Start with the badge gallery:

```text
badges/README.md
```

Pick one badge that feels inviting.

### 2. Choose a Tiny First Action

A first action can be simple:

- write a README
- make a safety checklist
- draw an idea
- write a reflection
- encourage someone
- save a small learning note

### 3. Save Gentle Evidence

Evidence can be private.

Evidence can be small.

Evidence does not need to be perfect.

## How to Choose a First Badge

Start with the badge that feels safest and most natural.

Good first choices:

- Seed Planter: if you want to create a Digital Garden
- Future Protector: if you want to learn online safety
- Joy Bringer: if you want to make someone smile
- Kindness Keeper: if you want to practice care
- Growth Guardian: if you want to reflect on progress

Badges are invitations, not tests.

## How Proof Works

Proof means evidence that something happened.

Proof may be:

- a reflection
- a screenshot
- a drawing
- a commit
- a trusted witness
- a photo
- a journal entry

Proof is checked for existence, not quality.

No one needs to rank it.

No one needs to grade it.

## How Witnesses Work

A witness confirms that the proof exists and matches the badge invitation.

A witness may be:

- yourself
- a friend
- a mentor
- a family member
- a teacher
- a JOYSPACE Guide
- a trusted community member

Witnesses do not judge quality.

Witnesses do not create authority.

## How Awards Are Celebrated

A badge award is a receipt of participation.

It may be celebrated with:

- a kind note
- a badge image
- a README update
- a private journal entry
- a family-only record
- a shareable gallery entry

Celebration should feel good, not stressful.

## Privacy and Safety Rules

- No personal data required.
- No passwords.
- No sensitive screenshots.
- No exact location, school, schedule, or private identifiers.
- Private proof is valid.
- Family-only proof is valid.
- Shareable proof requires consent.
- You may always keep your work private.
