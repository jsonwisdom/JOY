Purpose:

Help a newcomer answer four questions quickly:

1. Which quest do I start with?
2. Which quests are playable?
3. Which quests are complete or pending?
4. Where do I report confusion?

## How To Use This Log

Start with the lowest playable quest number.

If you get stuck, record where the path failed.

A failed replay is useful data.

Patch the doorway and replay.
