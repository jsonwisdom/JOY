Mathematical Rule

Inside repo must match outside public route

JoySpace is not green because a file exists. JoySpace is green only when the GitHub source, served public route, downloaded bytes, and human-visible page agree.

GREEN = HTTP_200 + SHA256(repo_source) == SHA256(public_body) + CONTENT_MARKER + PRIVACY_CHECK + HUMAN_VISUAL_CONFIRM
