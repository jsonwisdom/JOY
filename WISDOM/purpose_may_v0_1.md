# Purpose May v0.1

Status: SEEDED
Authority: false
Surface: Wisdom Family Game Night

## Purpose

Purpose May is the Game Night module for why the system should exist.

It teaches pause, care, and purpose before mechanics.

May is the porch question:

```text
May this move forward?
```

The answer is never automatic.

Purpose must be named.
Boundaries must hold.
Receipts must be ready.

## Core Question

```text
Why should this exist, and who does it help?
```

## Purpose Stack

```text
Care     -> does this protect people?
Clarity  -> can this be explained simply?
Consent  -> is sharing safe and allowed?
Receipt  -> what can be shown?
Replay   -> can it be checked again?
Rest     -> can we pause without penalty?
```

## May Rule

```text
May does not command.
May asks.
May pauses.
May protects the purpose.
May waits for receipts.
Authority remains false.
```

## Game Steps

```text
1. Pick one possible action.
2. Ask: why should this exist?
3. Ask: who might carry the cost?
4. Ask: what should stay off the public surface?
5. Ask: what receipt is needed next?
6. If unclear, choose REST_ALLOWED.
```

## Allowed Answers

```text
PURPOSE_CLEAR
RECEIPT_NEEDED
REST_ALLOWED
HOLD_FOR_OPERATOR
DO_NOT_ADVANCE
```

## Win Condition

```text
Everyone understands the purpose before anyone touches the tools.
```
