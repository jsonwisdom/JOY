# Wisdom Family Game Night V0.1

Status: ACTIVE
Authority: false
Membrane: HOLDS

## Purpose

Wisdom Family Game Night turns the family continuity spine into a safe, funny, replayable family activity.

It is designed for home use first.

It helps the family preserve memory, learn skills, tell stories, and understand receipts without turning family members into content, mascots, or repo objects.

## Core Rule

```text
Family first.
Fun second.
Receipts third.
Sharing only when safe.
Authority false always.
```

## Game Night Loop

```text
1. Pick a game.
2. Choose one safe memory, skill, lesson, or receipt.
3. Play for 10 to 30 minutes.
4. Save one small continuity note.
5. Decide: private, family-only, or shareable.
```

## Game 1: Continuity Bingo

Goal: make continuity visible.

Squares may include:

- saved a memory
- found a receipt
- learned a skill
- protected privacy
- helped someone understand the map
- told a family story
- labeled something private
- labeled something shareable
- found an old photo or artifact
- made someone laugh

Win condition:

```text
Everyone wins when the family story becomes easier to understand.
```

## Game 2: Receipt Hunt

Goal: teach proof without pressure.

Pick one safe artifact and answer:

- What is it?
- Who made it?
- When was it made?
- Where is it stored?
- Is it private, family-only, or shareable?
- Does it need a receipt?

Rule:

```text
No guessing. Receipts first.
```

## Game 3: Family Lore Cards

Goal: preserve wisdom in card form.

Each card may include:

- a lesson
- a saying
- a tradition
- a funny moment
- a value
- a skill
- a memory category

Privacy rule:

```text
Cards may preserve meaning without exposing private details.
```

## Game 4: Kindness Quest

Goal: make the system human.

Daily or weekly quest options:

- thank someone
- explain something simply
- preserve a memory
- help with a skill
- make a confusing thing easier
- choose privacy before posting

Win condition:

```text
Mrs Wisdom wins if the game creates kindness instead of stress.
```

## Game 5: Replay Race

Goal: reconstruct a story from receipts.

Steps:

1. Choose a harmless family story.
2. Gather safe clues.
3. Place the clues in order.
4. Mark unknowns as unknown.
5. Tell the story without pretending.

Rule:

```text
Unknowns are allowed. Fake certainty loses.
```

Source surface:

Lesson:

```text
3 + 3 + 3 = 9
9 Porches + 1 Jay = 10
```

The 9 are the Alabama Porch circle.

The 1 is Jay as witness/operator, not authority.

```text
Triad 1 -> MARYDEE / LEEANN / HEIDEE
Triad 2 -> JAYCEE / BREANN / BRELEE
```

Game steps:

```text
The math is memory language, not proof and not authority.
```

```text
Everyone can explain the map without turning anyone into a command role.
```

## Public Safety Filter

Before sharing anything, ask:

- Does this expose a private person?
- Does this reveal location, school, schedule, or personal identifiers?
- Does this turn a family member into content?
- Does this preserve context?
- Would the family understand and approve it?

If uncertain:

```text
Keep private.
```

## Scoreboard Rule

Score continuity, not human worth.

Good scores come from:

- preserving context
- protecting privacy
- making things understandable
- helping each other
- keeping receipts clean

Bad scores are never assigned to people.

Only systems, artifacts, or processes can fail.
