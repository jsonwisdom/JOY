# Family Vectors, Games, and Shareable Ideas V0.1

Status: ACTIVE
Authority: false
Membrane: HOLDS

## Purpose

This artifact turns the Wisdom Family continuity spine into safe, playful, shareable surfaces.

It exists for family use first.

It does not turn family members into mascots, products, repo objects, or proof claims.

## Core Rule

```text
Make it fun.
Keep it safe.
Preserve the receipt.
Protect the family.
```

## Family Vectors

### 1. Memory Vector

Purpose: preserve stories, milestones, small wins, lessons, and traditions.

Outputs:

- family story cards
- memory receipts
- milestone timelines
- yearly replay pages

### 2. Skill Vector

Purpose: help family members learn, build, and grow safely.

Outputs:

- GitHub garden cards
- coding quests
- art logs
- reading logs
- skill receipts

### 3. Game Vector

Purpose: make continuity playable without shame or scoring human worth.

Outputs:

- continuity bingo
- receipt hunt
- kindness quest
- replay race
- family lore cards

### 4. Protection Vector

Purpose: keep family safety visible before expansion.

Outputs:

- privacy checklist
- public/private label cards
- no-silent-drop checks
- family barrier audits

### 5. Share Vector

Purpose: create safe public surfaces that explain the project without exposing private life.

Outputs:

- shareable README snippets
- social captions
- one-page explainers
- printable family map

## Game Ideas

### Continuity Bingo

Players mark safe continuity actions:

- saved a memory
- checked a receipt
- learned a skill
- protected privacy
- helped someone understand the map

Win condition:

```text
Everyone wins when the family story becomes easier to understand.
```

### Receipt Hunt

Find one safe artifact and label it:

- what is it?
- who made it?
- when was it made?
- where is it stored?
- is it private or shareable?

### Kindness Quest

Daily micro-actions:

- tell someone what they did well
- help preserve a memory
- make a confusing thing easier
- choose privacy before posting

### Replay Race

A timed challenge to reconstruct a story from receipts only.

Rule:

```text
No guessing. Receipts first.
```

### Family Lore Cards

Create cards for values, sayings, traditions, and lessons.

Cards may describe meaning, not expose private details.

## Daily Workflow

```text
Scan.
Save.
Receipt.
Protect.
Replay.
Share only what is safe.
```

## Public Safety Filter

Before sharing anything, ask:

- Does this expose a private person?
- Does this reveal location, school, schedule, or personal identifiers?
- Does this turn a family member into content?
- Does this preserve context?
- Would the family understand and approve the surface?

If uncertain:

```text
Keep private.
```
